Uploading Daily Collection to Finacle from Handheld Machine (DSAUPLD)

Collection amount has to be credited to office accounts specific to that Agent through HTM Menu.
Verify the Transaction and open the Daily Collection Utility.
Enter the file path as C:\<Agent ID> and click the button Upload to PC
If upload is successful then the utility will display a successful message and create a file at agents folder at local disc C. The file format will be .trn
Change the file extension to .txt
Then login to Finacle and Invoke the Menu DSAUPLD and follow the below procedure.
DSA idEnter the Agent ID
Upload File NameClick on Browse Button and Access the file path C:\<Agent ID>. Select the applicable .txt file and press Open
Click on Submit Button and Note down the Tran ID
Access the menu HACLINQ and enter the office account to ensure that the transaction is proper.
