Account Closure Charge Collection (HCACC)

For collecting charges from an account while closing an account invoke the Menu HCACC and follow the below procedure.
FunctionO - Collect
A/c. IDEnter the Account Number or select using the search
Click on Go Button
The charges pending for collection will be displayed
Click on Submit

Verification of Charge Collection in an Account (To be done by another user having verification rights)

Invoke the Menu HCACC and follow the below mentioned procedure.
FunctionV - Verify
A/c. IDEnter the Account Number or select using the search
Click on Go button
The charges pending for collection will be displayed
Click on Submit. Charges will be automatically deducted from the account
The Transaction ID can be viewed using the Menu HACLINQ.
