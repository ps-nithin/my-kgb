Opening of Customer ID

Menu option CUMM
Function A-Add, Press F4 Other options are C - Copy, D - Suspend, I - Inquiry, L - List, M - Modify, U - Undelete, V - Verify, X - Cancel
Cust name Select using F2, in first filed, Press F4, enter name in second field, press F4
Staff If staff select Y and enter employee ID else enter N, press F4 If yes then name entered in Cust name field should be exactly which shown in EFI menu related to that staff number
Minor Enter Y/N, If y then enter details related to minor when system prompt, press F4
Non resident Enter Y/N. If Y enter NRE related details when system prompts, press F4
Address type Select: Default address to which communicatioins are to be sent by system, press F4 P - Permanent, C - Communication, E - Official
Address Enter address, Press F4
City Enter the city code, press F4 Postal code will automatically populate. Press F4 In HMISRPT report ID 102 provides help for city codes
Type Select appropriate customer type using F2, press F4
Group Select appropriate customer group using F2, press F4
Occupation Select occupation of the customer using F2, press F4
Constitution Select appropriate constitution using F2, press F4
Gender Enter M/F/O. Press F1 for help. Press F4
Nat.ID.Card.No Furnish distinguishing number of ID proof produced by the customer, press F4
Marital Status Select the marital status of the customer by using F2, press F4
Community Select appropriate community by using F2, press F4
Caste Select appropriate caste using F2, press F4
Cust. Rating Select using F2, press F4, As on: Furnish work date by pressing Ctrl+X, press F4
Tax slab Select appropriate tax slab from the list by pressing F2. For 15H, 15GFE, 15GMA, No TDS and BASE, Pan number is mandatory in PAN/GIR No. field and for Penal FORM60/FORM61 to be furnished in PAN/GIR No. field (Click on PAN/GIR No. field and enter value
Free Text 1 S/O, W/O Etc. (F1 will provide help). press F4
Free Text 2 Relative name. (F1 will provide help). press F4
Free Text 3 Enter age as on date. (F1 will provide help). press F4
Free Text 4 Enter annual income. (F1 will provide help). press F4
Free Text 5 Enter 12 digit Aaadhaar number of that customer, if available and customer desirese to accept Aadhaar based credits through this else enter NA. press F4
Free Code 1 Enter location code. press enter Entering 3 didgit SOL ID and pressing F2 will show all location defined for that SOL
Free Code 2 to 4 Remove default values. press F4 and select from list. press F4
Option code Enter E, press F4
Currency field Enter INR - press F4 and then F10
Note down 9 digit unique customer ID.

Modification of Customer ID

Menu option CUMM
Function M - Modify
Cust. ID Enter the customer ID to be modified and press F4. Select the required page to be modified by pressing F6 and press F4 & F10 after modifying the data

Verification of Customer ID

Menu opiton CUMM
Cust. ID Enter customer ID/select from the list by pressing F2. press F4, F6 seven times and press F4
Enter option Enter E, press F4, Ctrl+E, F4, F4, and F10

Extra info

Please note that as and when branch receives any information regarding the demise of a customer, then the relative customer ID should be modififed and after verification of modification, it should be suspended. For marking the customer as deceased, access menu CUMM >> Function "M" >> F4 >> Customer ID >> F4 >> Select status >> Enter "DEC" >> press F4, F10. After verification, it should be suspended using the function code "D"
