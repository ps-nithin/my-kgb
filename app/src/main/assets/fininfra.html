Introduction

Compared to Finacle 7, one of the major changes is in the Signature Management. In Finacle 10, a dedicated module named SVS is available in FININFRA for signature management. The module allows the user to Add/Update/Delete Signature and Photographs in an account or CIF ID. User can navigate to SVS module by accessing the following the below procedure
Select the Solution FININFRA
Click on Signature Verification System
Click on Signatures and Photographs
Click on Maintain Signatures and Photographs

Add Signature and Photograph

For adding signature to an account/CIF ID access the following Menu path and follow the procedure
Select the Solution FININFRA >> Signature Verification System >> Signatures and Photographs >> Maintain Signatures and Photographs
FunctionSelect Add
A/c. TypeNormal A/c will be auto populated
A/c. IDEnter the Account Number for which the signature is to be captured
CIF IDEnter the Customer ID
Click on Go Button
Signature Upload ModeSelect Upload from File and click the icon next to it
A new window will pop up. Click Browse button and select the relevant signature file to be uploaded
Click Ok
Image Access CodeSelect ALL
Signature Effective FromEnter the Current Date
Signature Expires OnEnter 31-12-2099
Click Submit Button
User will get the message “Signature and Photograph Record added Successfully. Verification is pending”

Verification of Signature and photograph

Select the Solution FININFRA >> Signature Verification System >> Signatures and Photographs >> Maintain Signatures and Photographs
FunctionSelect Verify
A/c. TypeNormal A/c will be auto populated
A/c. IDEnter the Account Number
CIF IDEnter the Customer ID
Click Go Button
Verify Pending Signatures and Photographs Tab will be displayed
The record will be displayed in a Grid, Click the icon under Signature Pop up window will display the signature. Click on the icon next to signature to enlarge the same.
Click Close Button
Click on Close Button
To verify the record, Tick the checkbox under Verify. (Note: To Reject the record tick the checkbox under Undo)
Click on Submit Button
User will get the message “Signature and Photograph Record(s) Verified Successfully"
