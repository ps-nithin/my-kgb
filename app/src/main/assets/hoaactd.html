Open Term Deposit Account (HOAACTD)

Invoke the Menu HOAACTD and follow the below procedure for opening Term Deposit Account for a Customer.
FunctionOpen
CIF IDEnter the CIF ID for which account needs to be created
Scheme CodeEnter the Scheme Code
Click on Go Button
General Details Tab
Mode of OperationSelect the appropriate option
Click on Validate
Interest & Tax Details Tab
Click on Validate. A pop up will be displayed. Click on Ok button
Scheme Details Tab
Deposit/Installment Amt.Enter the Deposit Amount
Deposit Period (Months/Days)Enter the Period in Months/Days
NominationSelect the appropriate option
Standing Instruction for Recurring Deposit (Applicable only for Recurring Deposit Schemes)
Debit A/c. ID Enterthe Standing Instruction Debit Account Number
Click on Validate
Visit Flow Tab
Renewal & Closure Tab
AutorenewalSelect the appropriate option, if applicable
Max Renewal Allowed / Renewals DoneEnter the Maximum no of times the deposit needs to be renewed (Applicable only if the Autorenewal option is selected as Limited)
Renewal Period (Months/Days)Enter the period for which the deposit needs to be renewed
Autorenewal SchemeWill be auto populated by the system
Autorenewal General ledger Sub HeadWill be auto populated by the system
Autorenewal Interest TableWill be auto populated by the system
Renewal OptionSelect the appropriate option
Nomination Details Tab (Applicable only if the Nomination is selected as Yes)
Registration NoWill be auto populated by the same. Note down the same.
CIF IDEnter the Nominee CIF .Applicable only if the nominee is having CIF
Nominee NameEnter the Name of the Nominee
RelationshipSelect the appropriate Relationship Code
Address Line 1Enter the Address details of the Nominee
Address Line 2Enter the Address details of the Nominee
Address Line 3Enter the Address details of the Nominee
CitySelect the applicable City Code
StateSelect the applicable State Code (66 for kerala)
CountrySelect the applicable country code (IN for India)
Postal CodeEnter the PIN Code
Nomination PcntEnter 100
Click on Validate
Related Party Details
The details of the main account holder will be displayed in the tab
If additional party needs to be added, Click on Add button or click the blue arrow on top right
Relation TypeSelect the appropriate option
Relation CodeSelect the appropriate option

If more than one additional party needs to be added, click on Add button or click the blue arrow on top right again and repeat the process
