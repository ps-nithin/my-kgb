Open Term Loan Account (EI and Non EI Type) (HOAACLA)

Invoke the Menu HOAACLA and follow the below procedure for opening Term Loan Account for a Customer
FunctionOpen
CIF IDEnter the CIF ID for which account needs to be created
Scheme CodeEnter the Scheme Code
Click on Go Button
General Details Tab
A/c. StatementSelect the appropriate option. If S - Statement/B - Both is opted enter the frequency as well
Mode of OperationSelect the appropriate option
Click on Validate
Loan Details Tab
Loan Amt.Enter the Loan Amount
Loan Period (Months/Days)Enter the Period in Months/Days
Repay MethodSelect the appropriate option
E - Recover up to Effective Amt (If Drawdown needs to be enabled)
N - No Batch Recovery (If Drawdown is not required)
Operative A/c. IDEnter the account from which the Demand need to be collected (Applicable only if Drawdown is opted)
LA Interest Tab (Applicable only if RPH is available)
Interest Deferred up to DateEnter the Repayment Holiday (RPH) End Date (If interest need to be postponed during Holiday Period)
Deferred/Applicable Interest RateSelect Application (If RPH is applicable)
Payment Plan Tab >> General
No. of InstallmentsEnter the Number of Installments (Excluding Holiday Period, If RPH is applicable)
Installment Start dateEnter the Installment Start Date
Interest Start DateEnter the Interest application Start Date
Installment FrequencySelect the appropriate repayment frequency.
Can be any of the following as applicable to the scheme
M - Monthly
Q - Quarterly
H - Half Yearly
Y - Yearly
Interest FrequencySelect the appropriate Interest application frequency
Can be any of the following as applicable to the scheme
M - Monthly
Q - Quarterly
H - Half Yearly
Y - Yearly
Payment Plan >> Holiday (Applicable only if RPH is available)
Normal Holiday Period (MM/Installments)Enter the RPH Period in Months
Interest During Holiday PeriodA - Apply
Interest Frequency during Holiday PeriodSelect the appropriate Interest application frequency
Can be any of the following as applicable to the scheme
M - Monthly
Q - Quarterly
H - Half Yearly
Y - Yearly
Payment Schedule Tab
Verify the schedule created. If the schedule needs to be corrected, visit Payment Plan Tab again and do the required modifications
Amortization ScheduleClick on Amortization Schedule to view the schedule details
Click on Print button, if Amortization Schedule Print is required, Else press Ok
MIS Codes Tab
Purpose of AdvanceSelect the appropriate option
Special ProgrammeSelect the appropriate option
Borrower TypeSelect the appropriate option
Click on Validate
Fees Tab
Click on validate
Account Limits Tab
Sanction DateEnter the Sanction Date of the account
Expiry DateWill be auto populated based on the date entered in Scheme Details Tab
Document DateEnter the applicable Date
Sanction LevelSelect the appropriate option
Sanction AuthoritySelect the appropriate option
Limit Ref. No.Enter the LAR No, if applicable
Drawing Power IndicatorSelect E - Equal
Click on Validate
Related Party Details
The details of the main account holder will be displayed in the tab
If additional party needs to be added, Click on Add button or click the blue arrow on top right
Relation TypeSelect the appropriate option
Relation CodeSelect the appropriate option
CIF IDEnter the CIF of the related party
If more than one additional party needs to be added, click on Add button or click the blue arrow on top right again and repeat the process
Click on Validate
Click on Submit and note down the A/c. ID
