Downloading of File to Handheld Machine from Finacle (DSADND)

Invoke the Menu DSADND and follow the below procedure.
FunctionWill be auto populated as DOWNLOAD
DSA Agent idEnter the Agent ID
Commit FlgSelect Y
Click on Submit Button. Message will be displayed as File Successfully Generated.
Download File through HTRFTOPC Menu
Invoke the Menu HTRFTOPC and follow the below procedure
Source FileClick on Search button
Select the file <Agent ID>.mst
Click on Submit Button
Click on Save Button. The file will get downloaded.
Click on Save As option and Save the file to the desired Agents Folder.
Open the Daily Collection Utility
A screen named Palmtec PC Data transfer Utility will be displayed.
Fill the source file column by clicking Small Square below Source File label and selecting the file created (Local disc C, in folder named agent ID). Please note the file must have an extension .mst
If the file name is displayed correctly then ensure the cord from the system is connected to the device properly.
Switch on the device.
Press F7 on the device. The device will display Transfer ?
Press Yes. The device will display a message Waiting for PC..
Click on the Download to Palmtec button seen on Palmtec PC Data transfer Utility.
If the download is successful then the utility will display a success message.
