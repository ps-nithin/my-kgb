Pass Book Printing (HPBP)

For Issuing passbook for an account, invoke the Menu HPBP and follow the below procedure.
A/c. IDEnter Account Number
New PassbookWill be auto populated
Start Line No.Will be auto populated
Start Page No.Will be auto populated
USer can modify the above values using Modify option
Click on Print
Print details will be displayed
Click on Print Print dialogue box will appear. Ensure proper printer is selected and Click on Print
Pass Book will be printed
