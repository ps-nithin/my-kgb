Introduction

In Finacle 7x all major transactions including Loan Disbursements were carried out using TM Menu. In Finacle 10, separate Menus are available for initiating each kind of transaction. Following are the Menus available in Finacle 10 for carrying out the transaction related activities.
Transaction Menus (Detailed workflow provided below)

Transaction in Operative Account (HTM)

This Menu is used for initiating all types of transactions in Operative Accounts (SBA,CCA,ODA,CCA).

Create a Cash Receipt Transaction in an Account

FunctionA - Add
Transaction Type/SubtypeSelect C/NR - Cash/Normal Receipt
Click Go Button
Part Transaction Type - Credit will be by default selected
A/c. IDEnter the Account Number
Ref. CCY/Amt.Enter the Transaction Amount
Transaction ParticularsEnter the transaction particulars (BY CASH will be auto populated)
Click on Save button and note down the Tran ID
Before posting/verifying the Transaction ID, the same can be modified using the M- Modify option.
For posting the transaction (To be done by another user having verification rights)
Invoke the Menu HTM and follow the below procedure.
FunctionP - Post
Transaction IDEnter the Tran ID
Click on Go Button
Verify the details entered
Click Submit
If the posting of the transaction is done at the time of creation the same needs to be Verified by another user. If the posting is done by another user then the transaction will be in verified status.

Inter Branch Cash Withdrawal

For initiating an Inter Branch cash withdrawal, invoke the Menu HTM and follow the below procedure.
To be done by the Branch initiating Cash Payment
FunctionA - Add
Transaction Type/SubtypeSelect C/NP - Cash/Normal Payment
Click Go Button
Part Transaction Type - Debit will be by default selected
A/c. IDEnter the BA -RE CASH CBS A/C 40XXX1023010003
Ref. CCY/Amt.Enter the Transaction Amount
Transaction ParticularsEnter the transaction particulars
Head Office Transaction TypeSelect P - Parking Transaction
OptionsSelect A- Additional Part Tran Details
Click Go Button
Category CodeSelect BACAS
Bank CodeWill be auto populated by the system
Branch CodeEnter the Branch Code to which the cash is being paid
Particulars 1Enter the Remarks
Amt 1Transaction Amount will be auto populated by the system
Advice Requisition Ind.Select S - System
Click Accept
Click on Save button and note down the Tran ID

FunctionP - Post
Transaction IDEnter the Tran ID
Click on Go Button
Verify the details entered
Click Submit
If the posting of the transaction is done at the time of creation the same needs to be Verified by another user. If the posting is done by another user then the transaction will be in verified status.

Granting of Instant TOD

For granting Instant TOD during a Cash Payment Transaction, Invoke the Menu HTM and follow the below procedure.
FunctionA - Add
Transaction Type/SubtypeSelect C/NP - Cash/Normal Payment
Click Go Button
Part Transaction Type - Debit will be by default selected
A/c. IDEnter the Account Number
Ref. CCY/Amt.Enter the Amount
Transaction ParticularsEnter the transaction particulars
Grant Temporary OverdraftSelect Yes
Grant DateWill be auto populated as Current Date
Expiry DateWill be auto populated as Current Date
Penalty DateWill be auto populated
Permitted BySelect the User ID using search
NotesEnter the Remarks
Click on Save button and note down the Tran ID
Before posting/verifying the Transaction ID the same can be modified using the M- Modify option
If the posting of the transaction is done at the time of creation the same needs to be Verified by another user. If the posting is done by another user then the transaction will be in verified status
