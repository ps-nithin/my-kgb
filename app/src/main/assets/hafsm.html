Introduction

For freezing and unfreezing the transactions in an account, HAFSM Menu needs to be used in Finacle 10. The main difference between Finacle 7 and Finacle 10 Freezing of an account is that multiple freeze reason codes can be added for an account. In Finacle 10, the menu option used for marking/lifting lien in an account is HALM. The detailed procedure for the same is furnished below:

Freeze an Account

To Freeze an account, Invoke the Menu HAFSM and follow the below procedure. Multiple Freeze reason codes can be linked for an account.
FunctionF - Freeze
A/c. IDEnter the Account Number
Freeze CodeSelect Debit Only (Currently only Debit Freeze is enabled for Branches)
Reason CodeSelect the appropriate reason using search
Click on Go Button
Details will be displayed in the Grid. Select the record by tick mark against the record
Multiple Freeze Reason Codes can be entered for an account by selecting the same against Freeze Reason Code
Click on Submit Button
Verification of the Freeze is to be done by another user having verification rights through HAFSM Menu.

Unfreeze an Account

FunctionU - Unfreeze
A/c. IDEnter the Account Number
Click on Go Button
Verify the details displayed.
Select the record by tick mark against the record
Click on Submit Button
Verification of the Unfreeze can be done by another user having verification rights through HAFSM Menu.
