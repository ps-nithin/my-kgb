DD Cancelation

For canceling a Demand Draft, Invoke the Menu HDDC and follow the below procedure.
FunctionC - Cancel
Demand Draft NoEnter the DD Number that need to be canceled
Demand Draft Issue DateEnter the date of DD
Transaction TypeSelect the appropriate option
Create Fee TransactionSelect Yes
Click on Go Button
The DD Details will be displayed
NotesEnter the Remarks, if any
Click on Submit and Note down the Transaction ID and Amount
Posting of the DD Payment is to be done using HDDC Menu. Before posting the Transaction ID, the same can be modified using the M - Modify option. If the posting of the transaction is done by the same user, the same needs to be Verified by another user using HTM Menu. If the posting is done by another user then the transaction will be in verified status.
