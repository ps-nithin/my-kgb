Introduction

In Finacle 7, Demand Draft Issuance and Payment was carried out using TM Menu. In Finacle 10, DD issuance is done using HDDMI and DD payment is done through HDDMP Menu. The following are the major DD Related Menus available in Finacle 10.
HDDMIFor issuing a Demand Draft
HDDMPFor Payment of DD
HDDPRNTTo Print a DD
HDDPALLTo Print all unprinted DDs
HDDICFor Inquiring Demand Draft issued
HDDIDFor Inquiring Demand Draft paid
HDDCTo Cancel a Demand Draft
HDDPDemand Draft Reports

Click on Submit and Note down the Transaction ID
Posting of the DD Issuance is to be done using HDDMI Menu. Before posting the Transaction ID, the same can be modified using the M- Modify option. If the posting of the transaction is done by the same user, the same needs to be Verified by another user using HTM Menu. If the posting is done by another user then the transaction will be in verified status.
