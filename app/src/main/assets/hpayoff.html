Close a Loan Account (HPAYOFF)

Invoke the Menu HPAYOFF and follow the below procedure for closing the Term Loan Account of a Customer.
FunctionP - Payoff Loan Amt
A/c. IDEnter the Account Number or select using the search
Pay Off Value DateCurrent date will be auto populated by the system
Transaction Type Select the appropriate option
C/CR - Cash Receipt
T/CI - Transfer Customer Induced
T/BI - Transfer Bank Induced
Collect/Refund A/c. IDEnter the account from which the amount needs to be debited (Applicable only if T/CI is selected)
Click on Go button
Click Accept after putting tick mark against the record that needs to be modified.
Loan Pay Off Process Tab will be displayed - Do not modify any values
Click on Create Transaction button
The transaction details will be displayed
Click on Submit Button
Click on Accept
