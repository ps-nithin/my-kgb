Reversal of Proxy Transactions

Posting of Proxy Transaction is to be done using the Menu HSPTM.
FunctionR - Reversal
StatusO - Outstanding
Tran. Date FromEnter the From Date
Tran. Date ToEnter the To Date
Include NullSelect No
Processing ModeSelect Manual
Click on Go Button
All the pending transactions will be displayed
Put tick mark against the accounts
Click on Submit
Note down the Transaction ID displayed against each Account and Click Ok
If the account is having sufficient balance, the transaction created will be in posted status and another user should verify the same through HTM Menu. If the account is not having sufficient balance then TOD needs to be granted.
