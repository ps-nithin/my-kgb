Deposit Receipt Printing (HDRP)

For printing Deposit Receipt, invoke the menu HDRP and follow the below procedure. For displaying all pending receipts for printing, invoke the Menu and Click on Go Button.
Users can use any of the following filter options to get the print of a specific account.
1. Scheme Code
2. CIF ID
3. From A/c. ID - To A/c ID
4. From A/c. Open Date - To A/c Open Date
The records pending for printing will be displayed in the Grid
Select the tick mark against the Account which needs to be printed
Press Accept
Deposit Receipt Print screen will be displayed
Print Srl. No.Ensure correctness of the Deposit Receipt Serial Number
Deposit Receipt Print IndicatorWill be auto populated as System Generated
Click Submit button
Print details will be displayed
Click on Print Print dialogue box will appear. Ensure proper printer is selected and Click on Print
Deposit Receipt will be printed
Click on Return button Desktop Print confirmation dialogue box will appear
If printed successfully, Click on Proceed
If the print is having any issues, untick all the check boxes and click on proceed to Go Back to Printing.
