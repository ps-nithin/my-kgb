Term Deposit Subsidy (TDSUBSDY)

For linking Term Deposit subsidy to a loan account, Invoke the menu HTDSUBSDY and follow the below Procedure.
Before linking Term Deposit as Subsidy to a loan account ensure the following
Term Deposit should be having Interest Table code as ZEROT
ModeAdd
Loan Account NoEnter the Loan Account Number to which the subsidy needs to be linked
Term Deposit Account NoEnter the Term Deposit Subsidy Account Number
Valid Upto DateWill be auto populated by the system
Click Submit
Before verification, lien has to be marked against the Term Deposit through HALM Menu. Verification of the same is to be done by another user having verification rights through HTDSUBSDY Menu.
