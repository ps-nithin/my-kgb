Printing a DD

For printing a Demand Draft, invoke the menu HDDPRNT and follow the below procedure.
Tran./Part Tran. IDEnter the Transaction No and Serial Number
Issue DateEnter the Current Date
Click on Go Button
DD Print screen will be displayed
Inventory TypeWill be auto populated
Inventory NoEnsure correctness of the Demand Draft Serial Number
Click Submit button
Print details will be displayed
Click on Print Print dialogue box will appear. Ensure proper printer is selected and Click on Print
DD will be printed
Click on Return button Desktop Print confirmation dialogue box will appear If printed successfully, Click on Proceed If the print is having any issues, untick all the check boxes and click on proceed to Go Back to Printing.
