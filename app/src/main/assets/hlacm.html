Unlinking of Collateral while Closing LTD Accounts

If the LTD Account is opened in Finacle 10, the collateral ID is to be unlinked through HLACM Menu.
Unlinking of collateral from LTD accounts created in Finacle 7 (Migrated Accounts) is to be done using HSCLM Menu. For unlinking the security ID through HSCLM follow the procedure detailed above.
For unlinking Term Deposit collateral opened in Finacle 10, invoke the Menu HLACM and follow the below procedure.
FunctionM - Modify
A/c. IDEnter the Account Number
Click on Go Button
The Term Deposit Account Number and Security details will be displayed
RecordSelect the Tick mark against Del box to unlink the Deposit from Loan Account
Click on Submit Button
Verification of the collateral linking can be done by any user (other than creator) having verification rights through HLACM Menu.
