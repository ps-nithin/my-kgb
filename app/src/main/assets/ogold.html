Open Gold Loan Accounts (OGOLD)

This Menu is used for opening Gold Loan Accounts. Invoke the Menu OGOLD and follow the below mentioned procedure.
Key Fields
ModeA - Add
Scheme CodeSelect the applicable Scheme
NominationSelect the appropriate option
CIFSelect the appropriate option
CIF IDEnter the CIF ID (Applicable if Enter CIF option is selected)
Account NoEnter the Account Number (Applicable if Enter Fetch from Account option is selected)
Click Go Button
Basic Information
Ornament DetailsEnter the Ornament Details. 10 Fields are available for entering the details
No of ItemsEnter the No of Ornaments pledged against each item
Gross Weight (in Gms)Enter the Gross Weight of the ornaments
Net Weight (in Gms)Enter the Net Weight of the ornaments
Appraised ValueEnter the Appraised Value
Loan AmountEnter the Loan Amount
PurposeEnter the Purpose of Advance
Borrower TypeSelect the appropriate option
Land Holding (Applicable only if the Scheme selected is AGL)
Land HoldingEnter the land holding area and select the appropriate area type
Survery NoEnter the Survey No
Nominee (Applicable only if Nomination is opted in Key Field)
Nominee CategorySelect the Appropriate option
Nominee CIFEnter the Nominee CIF Applicable If nominee is having CIF
Nominee NameEnter the Name of the Nominee
Address Line 1Enter the Address details of the Nominee
Address Line 2Enter the Address details of the Nominee
City CodeSelect the applicable City Code
RelationshipSelect relationship of Nominee to the Customer
Nominee Date of Incorporation
Nominee Guardian (Applicable only Nomination is opted as Nominee is Minor in Key Field)
Guardian CategorySelect the Appropriate option
Guardian CIFEnter the Guardian CIF ( Applicable only If the Guardian is having CIF)
Guardian NameEnter the Name of the Guardian
Address Line 1Enter the Address details of the Guardian
Address Line 2Enter the Address details of the Guardian
City CodeSelect the applicable City Code
Guardian Type/RelationSelect the applicable Guardian Relationship
Transaction
Transaction ModeSelect Transfer
Transaction AmountEnter the Disbursement Amount
Transfer Account NoEnter the operative account of the customer
Print/FATCA
Print Opening Form?Select Yes, If Print is required at the time of creation. Else select No
FATCA Applicable?Select the appropriate option

Before verification, modification can be done by any user using the menu OGOLD. Verification of the record to be done by another user having verification rights using the same Menu.
