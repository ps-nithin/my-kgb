Introduction

In Finacle 7, all types of Standing Instruction were created using the Menu SIM. In Finacle 10, Standing Instruction transactions can be created using HSIM and HSSIM Menu.The following are the major Standing Instruction related Menus available in Finacle 10.
HSSIMSimple Standing Instruction Maintenance
HSIMStanding Instruction Maintenance
HSIRPStanding Instruction Register Print
HSIETRStanding Instruction Executed Today Report

Simple Standing Instruction Maintenance

For creating a Simple Standing Instruction, Invoke the Menu HSSIM and follow the below procedure.
FunctionAdd
Click on Go Button
Header Details Tab
Standing Instruction TypeSelect the appropriate option
CIF IDEnter CIF ID of the Debit Account (Applicable only for Customer Induced Transaction)
Execution Details
Standing Instructions FrequencySelect the appropriate SI frequency
Execution TimeSelect the appropriate option
Next Execution DateEnter the date of Next Execution
End DateEnter the SI End Date
Validate CCY HolidaySelect Yes
Del. Transaction if not PostedSelect the appropriate option
AutopostSelect the appropriate option
Carry Forward If FailedSelect the appropriate option. (Select Y to carry forward the SI for failed execution)
Carry Forward Max. TimesEnter the number of times the SI needs to be carried forward
in case of failed execution. Note: If 0 is entered, the SI will be indefinitely carry forwarded
Instruction Details Tab
Create Memo Pad EntrySelect No
CCYEnter INR
A/c. IDEnter the Debit Account Number
Debit/CreditSelect Debit
Amt. TypeSelect F - Fixed
Amt.Enter the Amount
RemarksEnter Remarks, if any
Click Add Button on the top left to enter Credit Account Details
Create Memo Pad EntrySelect No
CCYEnter INR
Debit/CreditSelect Credit
A/c. IDEnter the Credit Account Number
Amt. TypeSelect F - Fixed
Amt.Enter the Amount
RemarksEnter Remarks, if any
Click on Submit Button. Note down the SI Serial Number
Verification of the Simple Standing Instruction needs to be done using HSSIM Menu. Before verification the same can be modified using the M- Modify option in HSSIM.
