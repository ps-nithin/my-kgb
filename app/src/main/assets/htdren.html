Term Deposit Renewal (HTDREN)

For renewal of a matured Term Deposit Account Invoke the Menu HTDREN and follow the below procedure.
FunctionR - Renewal
A/c. IDEnter the Account Number
Click on Go Button
The Effective Open DateMaturity Date of the deposit will be auto populated
Calculate Overdue IntSelect the appropriate option. Select Yes if the account is overdue, Else Select No
Overdue Int. CodeApplicable code will be auto populated
Renewal OptionSelect the appropriate option
Renewal Interest Table CodeWill be auto populated by the system (Change the code if applicable)
Renewal Scheme CodeExisting scheme code will be auto populated. User can change the Renewal Scheme Code, if applicable
Renewal Period (MMM/DDD)Enter the period for which the deposit needs to be renewed
Renewal CCYINR will be auto populated
Renewal General Ledger Subhead CodeWill be auto populated by the system
Renewal CCY/Amt.Enter the Renewal Amount. (Applicable only if the renewal option selected is F - Fixed Amt.)
Additional CCY/Amt.Enter the Additional Amount to be added up (Applicable only if the renewal option selected is A - Additional Amt. or E - Principal + Extra)
Source A/c. for Additional Amt.Enter the Source Account for Additional Amount (Applicable only if the renewal option selected is F - Fixed Amt. or A - Additional Amt. or E - Principal + Extra)
Print renewal ConfirmationSelect Later
Click on Submit
