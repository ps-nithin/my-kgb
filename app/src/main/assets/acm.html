Modification of SB/CA account after verification
Menu option ACM
Function Enter M and Press F4 Other options are I - Inquiry, V - Verify, X - Cancel
A/c ID Enter the account number to modified, Press F4
Enter option Enter relative option code by selecting from the list on pressing F2 and after required modification press F4 & F10 0 (Zero) - general details, S-Scheme details, N-Nomination details, A-Related party details, R-NRE details

Verification of modification in SB/CA accounts

Menu option ACM
Function Enter V and press F4
A/c ID Enter the account number/select from the list by pressing F2 and press F4
Option code Enter all the relative option codes in which data modified one by one and press F4 and F10

Extra info

For changing the authorized signatories, after entering option A, user has to select the page containing details of signatories to be changed by pressing 'down arrow' key and select Del. Flag field and modify the existing value N to Y and enter the details of new signatory in a new page by pressing down arrow.
