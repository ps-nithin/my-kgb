Payment of DD

For payment of a Demand Draft, Invoke the Menu HDDMP and follow the below procedure.
FunctionAdd
Demand Draft A/c. IDEnter SMGDD
Payee Details
Transaction TypeSelect the appropriate option
Payee A/c. IDEnter the Account Number (Applicable only if the Transaction Type is Transfer)
Payee NameEnter the Payee Name (Applicable only if the Transaction Type is Cash)
CCYINR (Applicable only if the Transaction Type is Cash)
Instrument TypeSelect DDS from the search
Click on Go button
Demand Draft No.PayeeEnter the DD number in the second column and click on Tab
The DD Details will be displayed
NotesEnter the Remarks, if any
Click on Submit and Note down the Transaction ID
Posting of the DD Payment is to be done using HDDMP Menu. Before posting the Transaction ID, the same can be modified using the M - Modify option. If the posting of the transaction is done by the same user, the same needs to be Verified by another user using HTM Menu. If the posting is done by another user then the transaction will be in verified status.
