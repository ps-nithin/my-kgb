Introduction

In Finacle 7, all types of account closure (SBA,CAA,TDA,ODA,CCA,LAA) was carried out using the Core Menu CAAC and the verification of the same was done through CAACAU Menu. In Finacle 10, the core menus used for account closure have been splitted, for closing each type of account, different menus need to be invoked. The following are the core Menus that need to be used for closing the accounts.
HCAACClosure of all type of Operative Accounts (CASA and CCOD)
HCAACTDClosure of Term Deposits
HPAYOFFClosure of Term Loan Accounts
CAACLAFinal Closure of Loan Accounts
The detailed workflow for closing different types of accounts are given below.

Close an Operative Account (HCAAC)

Invoke the Menu HCAAC and follow the below procedure for closing the Operative Account (SBA,CAA,CCA,ODA) of a Customer.
FunctionZ - Close
A/c. IDEnter the Account Number or select using the search
BalancePut Tick Mark against transfer
Transaction Type
Transfer A/c. IDEnter the account number to which the balance needs to be credited
Click on Go button

Verify the details displayed
Closure Tab

Click Submit. Note down the Tran ID
