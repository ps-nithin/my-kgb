Step 1 - SOL Validation

Invoke the Menu CSOLVAL and follow the below Procedure
FunctionSelect V - Validation Run
Click on Submit
System will check whether any pending entries exist for the SOL and the Report of blocking validations will be generated, if any. Invoke the Menu HPR and view the Report. If any pendencies exist, clear the Same. If the SOL validation is successful, the following message will be displayed in the Report
40XXX - CSOLVAL validated successfully for the day DD-MM- YYYY by USERID. Kindly Submit the SOL.

Step 2 - SOL Submission (To be done by an Officer)

Invoke the Menu CSOLVAL and follow the below Procedure
FunctionSelect S - Submission
Submission RemarksEnter the remarks in the following format
For DD-MM-YYYY (Eg: IF EOD is done for 15-01-2024 Enter the remarks as FOR 15-01-2024)
Click on Submit
Invoke the Menu HPR to view the report generated. If the SOL is submitted Successfully the following message will be displayed.
40XXX - CSOLVAL Submitted successfully for the day DD-MM- YYYY by USERID
If the above message is displayed, the EOD activity of the branch is completed. Remaining activities will be done centrally by the Functional Team.
