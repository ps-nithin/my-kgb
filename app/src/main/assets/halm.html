Lien Marking (HALM)

Lien Marking in an Account
For marking user lien in an account Invoke the Menu HALM and follow the below procedure.
FunctionAdd
A/c. IDEnter the Account Number
Module TypeU - User Defined
Click on Go Button
New Lien AmountEnter the Lien Amount
From Effective DateWill be auto populated by the system
Lien ReasonSelect the appropriate reason using search
NotesEnter the remarks
More than one Lien Reason Code can be added by clicking on Add button
Click on Submit Button
Verification of the Lien marking can be done by any user having verification rights through HALM Menu.

Unmarking lien in an Account

For unmarking user lien in an account Invoke the Menu HALM and follow the below procedure.
FunctionM - Modify
A/c. IDEnter the Account Number
Module TypeU - User Defined
Click on Go Button
New Lien AmountModify the amount as zero
NotesEnter the remarks for unmarking the lien
Click on Submit Button
Verification of the Lien unmarking can be done by any user having verification rights through HALM Menu.
