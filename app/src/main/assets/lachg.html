Loan Charge Collection (LACHG)

For collecting charges from a Term Loan Account (LAA Type), invoke the Menu LACHG and follow the below procedure.
ModeAdd
Loan Account NoEnter the Loan Account Number
Click Go Button
Loan Account Details
Scheme CodeWill be auto populated
NPA StatusWill be auto populated
Transaction Details
Commission Acct NoSelect the appropriate Account
ChargeEnter the charge amount to be collected
IGST Applicable?Select the appropriate option
Tran ParticularsEnter the Transaction Remarks
NPATC Ref NoIf NPA Account, Enter the NPATC Reference Number
Value DateWill be auto populated as Current Working Date
Click on Fetch Data
Transaction Summary
GST AmountWill be auto populated
Cess FeeWill be auto populated
Total Debit AmountWill be auto populated
Click on Submit. Note down the Reference Number
Before verification the same can be modified using the M- Modify option in LACHG. Verification of the record needs to be done using the same Menu.
