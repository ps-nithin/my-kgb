Linking/Unlinking of Collateral ID to Loan Accounts (HSCLM)

This menu is used for performing the following operations.
1. LINK/MODIFY/UNLINK -Inventory (Stock)/ Book Debts with loan accounts
2. UNLINK-Deposit Collateral ID linked to loan against deposit opened before migration and linked through HSCLM.
3. If loan against deposit (LTD) is opened post migration same to be unlinked via HLACM menu
4. LINK/MODIFY/UNLINK-Deposit collateral ID created and linked through HCLM/HSCLM for loans other than LTD

Link/Modify Collateral ID to an Account

Invoke the Menu HSCLM and follow the below procedure
FunctionL - Link
A/c. IDEnter the Account Number
Collateral IDEnter the Collateral ID
Click Accept Button
Apportioned ValueEnter the collateral Value
NatureSelect the applicable nature of security
Click on Submit and note down the Collateral ID and the Account Number linked
Verification of the collateral linking can be done by any user (other than creator) having verification rights through HSCLM Menu
Post Verification, Modification of the collateral linkage can be done by any user by invoking the Menu HSCLM and the same to be verified.

Unlinking a Collateral ID from Account

Invoke the Menu HSCLM and follow the below procedure
FunctionU - Unlink
A/c. IDEnter the Account Number
Collateral IDEnter the Collateral ID
Click Accept Button
Reason CodeSelect the Reason Code
Click on Submit. The Collateral ID will be delinked from the Account.
Verification of the collateral linking can be done by any user (other than creator) having verification rights through HSCLM Menu
Before Verification of the Collateral ID the same can be canceled through HCLM Menu.
