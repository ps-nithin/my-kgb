Pass Sheet Printing (HPSP)

For generating the Pass Sheet for an account invoke the Menu HPSP and follow the below procedure.
Statement TypeSelect Statement of A/c
EventBy default Duplicate will be selected
From A/c. IDEnter the Account Number
To A/c. IDWill be auto populated
Period FromEnter the Start Date for Statement
Period ToEnter the End Date for Statement
Print >> WaiveFees If Charges are to be waived, put tick mark
By default charge will be collected for the first pass sheet also. If the charges need to be waived, select the option accordingly.
Click Submit
Invoke the Menu HPR to view the passsheet. Select the Print Screen option to print the same.
